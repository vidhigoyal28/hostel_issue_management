Customer Segmentation / Complaint Analysis System

A Machine Learning project that analyzes and segments customer/hostel complaints using data preprocessing, clustering, and visualization techniques. This project helps in identifying patterns and improving decision-making.


 Features

- 📊 Data preprocessing and cleaning
- 📈 Exploratory Data Analysis (EDA)
- 🤖 Machine Learning models for segmentation
- 📌 Clustering using algorithms like K-Means
- 📉 Visualization of clusters and insights
- 🧹 Handling imbalanced dataset

 Tech Stack

- Python 🐍
- Pandas & NumPy
- Matplotlib & Seaborn
- Scikit-learn
- Jupyter Notebook
